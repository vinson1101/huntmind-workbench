# TalentFlow Skill

`TalentFlow` is the resume-processing skill inside `huntmind-workbench`.

Responsibilities:
- ingest resumes from local files or temporary folders
- normalize candidate payloads
- build `batch_input.json`
- validate and sanitize HuntMind outputs
- generate final reports and owner summaries

Non-responsibilities:
- company industry classification
- JD to template routing ownership
- taxonomy/template/hints authority
- model selection, memory, or agent identity

Those decision-enhancement capabilities belong to HuntMind or the paid enhancement layer.
