## 不可接受的错误类型

1. **套分/批量同分**：多个候选人 dimension_scores 完全相同，或分数仅因名字不同而不同
2. **dimension_evidence 写占位文本**：如"hard_skill_match维度评估"
3. **core_judgement 分数与 total_score 严重脱节**：如写 82 但 total_score 是 56
4. **reason/risk/action 整批只换名字**：无候选人差异化描述
5. **action_timing 滥用 today**：maybe/B类默认 today
6. **候选人在 A/B/C 档间大幅错位**：如真实 A 类被排在 C
